hiperwalk.QuantumWalk#

class hiperwalk.QuantumWalk(graph=None, adjacency=None, **kwargs)[source]#

Basic (abstract) class for Quantum Walks.

Basic methods and attributes used for implementing specific Quantum Walk models.

Parameters:
graph

Graph on which the quantum walk occurs. It can be the graph itself (hiperwalk.graph.Graph) or its adjacency matrix (scipy.sparse.csr_array).

adjacencyscipy.sparse.csr_array, optional

Deprecated since version 2.0a1: It will be removed in version 2.0. Use graph instead.

The adjacency matrix.

Raises:
TypeError

if adj_matrix is not an instance of scipy.sparse.csr_array.

Warns:
If adjacency is set. It is deprecated. Use graph instead.

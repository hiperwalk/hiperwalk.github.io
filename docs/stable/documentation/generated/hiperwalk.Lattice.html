hiperwalk.Lattice#

class hiperwalk.Lattice(adj_matrix)[source]#

Abstract class for Lattice graphs.

Lattice graphs are embedabble in a Euclidian Space, forming a regular tiling.

Notes

Given an embedding, directions can be assigned to the arcs.

hiperwalk.ContinuousTime.set_time#

ContinuousTime.set_time(time=None, **kwargs)[source]#

Sets time.

Sets time and generates the evolution operator corresponding to the specified time.

Parameters:
timefloat
** kwargs

Additional arguments for updating the evolution operator. For example, whether to use neblina HPC or not. See set_evolution() for more options.

Raises:
ValueError

If time < 0.

See also

set_evolution

Notes

The evolution operator is given by

\[U(t) = e^{-\text{i}tH},\]

where \(H\) is the Hamiltonian, and \(t\) is the time.
